Example Domain is a domain name that is reserved for use in documentation and illustrative examples. It is maintained by the Internet Assigned Numbers Authority (IANA) and is intended to be used in examples in documents, tutorials, and other educational materials.

You may use this domain in literature, training materials, or other forms of documentation without prior coordination or asking for permission.

Additional Information

  • Example Domain is not intended for actual deployment of websites or applications.
  • It helps prevent accidental conflicts or misuse in sample code or teaching.
  • More details can be found at IANA: Example Domains.
